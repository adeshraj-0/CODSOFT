# CODSOFT
# 🛍️ Origin Store

Origin Store is a modern and responsive E-Commerce Web Application developed as part of my **Web Development Internship at CODSOFT**.

This project was built to gain hands-on experience in React development, Firebase Authentication, routing, cart management, form validation, PDF invoice generation, and application deployment.

## 🚀 Live Demo

Live Website: https://codsoft-2kxj.vercel.app/

## ✨ Features

- 🛍️ Product Listing
- 🔍 Product Search Functionality
- 📦 Product Details Page
- 🛒 Shopping Cart
- 💾 Persistent Cart using Local Storage
- 🔐 Firebase Email and Password Authentication
- 👤 User Signup, Login and Logout
- 🛡️ Protected Checkout Page
- 🛡️ Protected Order Success Page
- 💳 Checkout Form with Validation
- 📄 Professional PDF Invoice Generation
- ⬇️ Download Invoice as PDF
- 📱 Responsive User Interface

## 🛠️ Tech Stack

- React.js
- Vite
- JavaScript
- CSS
- Firebase Authentication
- React Router DOM
- jsPDF
- jsPDF AutoTable
- React Icons
- GitHub
- Vercel
